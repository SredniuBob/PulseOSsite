Документация PulseOS

Полное руководство по установке, настройке и использованию дистрибутива PulseOS на базе Ubuntu 24.04.03 LTS

Примечание

Эта документация постоянно обновляется. Последнее обновление: 15 ноября 2023 г. Для PulseOS версии 1.0 "Noble Narwhal".

Введение в PulseOS

PulseOS — это дистрибутив Linux, основанный на Ubuntu 24.04.03 LTS (Noble Numbat). Он сочетает в себе стабильность Ubuntu с современным интерфейсом и оптимизациями для повышения производительности.

Ключевые особенности

  • Основан на Ubuntu 24.04.03 LTS — долгосрочная поддержка до апреля 2027 года
  • Современная среда рабочего стола — GNOME 46 с пользовательскими темами и расширениями
  • Оптимизация производительности — улучшенное управление памятью и временем загрузки
  • Предустановленные приложения — полный набор программ для работы и мультимедиа
  • Поддержка аппаратного обеспечения — современные драйверы и поддержка новых устройств

Системные требования

Компонент Минимальные Рекомендуемые
Процессор 2 ядра, 1.5 ГГц 4+ ядер, 2.0+ ГГц
Оперативная память 2 ГБ 8 ГБ или больше
Хранилище 25 ГБ 50 ГБ (SSD рекомендуется)
Видеокарта Поддержка OpenGL 3.3 Поддержка Vulkan 1.1

Установка PulseOS

PulseOS может быть установлен как основная система или вместе с другой ОС (двойная загрузка).

Создание загрузочного носителя

  1. Скачайте образ PulseOS с официального сайта
  2. Подготовьте USB-накопитель объемом не менее 8 ГБ
  3. Используйте одну из программ для записи образа:
    • BalenaEtcher (Windows, macOS, Linux)
    • Rufus (Windows)
    • dd команда (Linux/macOS)

Процесс установки

После загрузки с USB-носителя следуйте инструкциям установщика:

Важно

Если вы планируете установку рядом с Windows, отключите "Быстрый запуск" в Windows перед установкой. Также рекомендуется сделать резервную копию важных данных.

Шаги установки:

  1. Выберите язык системы
  2. Настройте раскладку клавиатуры
  3. Подключитесь к сети Wi-Fi или Ethernet
  4. Выберите тип установки:
    • Обычная установка (рекомендуется)
    • Минимальная установка
    • Установка с шифрованием LUKS
  5. Разметка диска (автоматическая или ручная)
  6. Настройка пользователя и пароля
  7. Дождитесь завершения установки и перезагрузите систему
sudo apt update && sudo apt upgrade

После установки рекомендуется обновить систему до последних версий пакетов.

Первые шаги после установки

Обновление системы

После первой загрузки откройте Терминал и выполните:

# Обновление списка пакетов
sudo apt update

# Установка обновлений
sudo apt upgrade

# Обновление ядра (если доступно)
sudo apt dist-upgrade

# Очистка кэша пакетов
sudo apt autoclean
sudo apt autoremove

Установка проприетарных драйверов

Для видеокарт NVIDIA или AMD рекомендуется установить проприетарные драйверы для лучшей производительности:

# Просмотр доступных драйверов
ubuntu-drivers devices

# Автоматическая установка рекомендованных драйверов
sudo ubuntu-drivers autoinstall

# Или ручная установка конкретной версии
sudo apt install nvidia-driver-535

Настройка часового пояса и локализации

# Настройка часового пояса
sudo timedatectl set-timezone Europe/Moscow

# Настройка локализации (русский язык)
sudo localectl set-locale LANG=ru_RU.UTF-8
sudo localectl set-keymap ru
Внимание

После изменения локализации потребуется перезагрузить систему для применения всех настроек.

Системные настройки

PulseOS использует среду рабочего стола GNOME с пользовательскими расширениями. Основные настройки доступны через приложение "Параметры".

Настройки внешнего вида

  • Темы: PulseOS включает светлую и темную тему, которые можно переключать в настройках
  • Обои рабочего стола: В коллекции доступны обои PulseOS и возможность установки собственных
  • Экран блокировки: Настройка фона и времени до блокировки экрана

Настройки энергосбережения

Для ноутбуков особенно важны настройки энергосбережения:

# Установка утилит для управления питанием
sudo apt install tlp tlp-rdw

# Запуск службы TLP
sudo systemctl enable tlp
sudo systemctl start tlp

# Проверка статуса
sudo tlp-stat -s

Настройки принтера

PulseOS использует систему печати CUPS. Для настройки принтера:

  1. Откройте "Параметры" → "Принтеры"
  2. Нажмите "Добавить принтер"
  3. Система автоматически обнаружит сетевые принтеры
  4. Для USB-принтеров подключите устройство и следуйте инструкциям

Или используйте терминал:

# Запуск веб-интерфейса CUPS
sudo systemctl start cups
# Откройте http://localhost:631 в браузере

Управление пакетами с APT

APT (Advanced Package Tool) — основной инструмент управления пакетами в PulseOS. Он работает с репозиториями .deb пакетов.

Основные команды APT

Команда Описание
sudo apt update Обновить список доступных пакетов
sudo apt upgrade Установить обновления для всех пакетов
sudo apt install <пакет> Установить пакет
sudo apt remove <пакет> Удалить пакет (сохраняя конфигурацию)
sudo apt purge <пакет> Полностью удалить пакет с конфигурацией
apt search <запрос> Поиск пакетов
apt show <пакет> Показать информацию о пакете

Добавление репозиториев

Для добавления сторонних репозиториев:

# Добавление репозитория
sudo add-apt-repository ppa:repository-name

# Или ручное добавление в /etc/apt/sources.list.d/
echo "deb http://repository.example.com/ubuntu noble main" | sudo tee /etc/apt/sources.list.d/repository-name.list

# Добавление ключа репозитория
sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys KEY_ID
Безопасность

Добавляйте только доверенные репозитории. Неподписанные репозитории могут содержать вредоносное ПО.

Навигация в файловой системе

Команда Описание Пример
pwd Показать текущую директорию pwd
ls Список файлов ls -la
cd Сменить директорию cd ~/Documents
mkdir Создать директорию mkdir new_folder
rmdir Удалить пустую директорию rmdir old_folder

Работа с файлами

Команда Описание
cp Копирование файлов
mv Перемещение/переименование
rm Удаление файлов
cat Просмотр содержимого
nano/vim Текстовые редакторы
chmod Изменение прав доступа
chown Изменение владельца

Полезные сочетания клавиш в терминале

  • Ctrl+C — прервать выполнение команды
  • Ctrl+D — завершить сессию (выход)
  • Ctrl+L — очистить экран
  • Tab — автодополнение
  • Стрелки вверх/вниз — история команд
  • Ctrl+R — поиск в истории команд

Решение общих проблем

Проблемы с загрузкой

Если система не загружается, попробуйте следующие шаги:

  1. При загрузке нажмите Esc для выбора ядра
  2. Выберите режим восстановления (Recovery Mode)
  3. Используйте опцию "root" для доступа к терминалу
  4. Проверьте файловую систему: fsck -f /dev/sdX
  5. Переустановите ядро: apt install --reinstall linux-image-generic
  6. Обновите конфигурацию загрузчика: update-grub

Проблемы с Wi-Fi

# Проверка состояния Wi-Fi
sudo systemctl status NetworkManager

# Перезапуск сетевого менеджера
sudo systemctl restart NetworkManager

# Просмотр доступных сетей
nmcli device wifi list

# Подключение к сети
nmcli device wifi connect "SSID" password "пароль"

# Просмотр информации о драйверах Wi-Fi
lspci -k | grep -A 3 -i network

Проблемы со звуком

# Проверка установленных драйверов звука
lspci -v | grep -A 7 -i "audio"

# Перезапуск звуковой системы
systemctl --user restart pulseaudio
# Или для PulseAudio 15+
pulseaudio -k && pulseaudio --start

# Проверка уровней звука
alsamixer
Поддержка

Если вы не можете решить проблему самостоятельно, обратитесь за помощью на форум сообщества PulseOS или создайте issue на GitHub репозитории проекта.
